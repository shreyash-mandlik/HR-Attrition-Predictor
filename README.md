# HR Employee Attrition Predictor 👥

A machine learning web app that predicts whether an 
employee is likely to leave the company.

## 🌐 Live Demo
👉 [Try the app here!](https://hr-attrition-predictor-gtlvla5zncmcgnau9rhkep.streamlit.app/)

## 🎯 Problem Statement
Employee attrition costs companies millions every year. 
This app predicts which employees are at high risk of 
leaving so HR can take preventive action.

## 🛠️ Tech Stack
- Python
- Scikit-learn (Random Forest)
- Streamlit
- Pandas, NumPy
- Matplotlib, Seaborn

## 📊 Key Findings from EDA
| Factor | Attrition Rate |
|--------|---------------|
| With Overtime | 30.5% |
| Without Overtime | 10.4% |
| Low Job Satisfaction | 22.8% |
| High Job Satisfaction | 11.3% |

## 📈 Model Performance
- Accuracy: 87.41%
- Algorithm: Random Forest (balanced class weights)
- ROC AUC Score: 0.728

## 🔍 Top Factors Affecting Attrition
1. Monthly Income (8.0%)
2. Overtime (7.0%)
3. Age (6.3%)
4. Distance From Home (4.0%)
5. Years At Company (4.8%)

## 💡 Business Recommendations
- Reduce overtime hours
- Increase salaries for low paid employees
- Improve job satisfaction
- Consider remote work for employees far from office
- Pay extra attention to younger employees

## 📁 Dataset
IBM HR Analytics Employee Attrition Dataset
- 1470 employees
- 35 features
